util_json

Безопасная работа с JSON.

index.py

loads_safe(s, default=None, max_bytes=2000000)
Парсит JSON строку/bytes. При ошибке возвращает default вместо исключения.
Проверяет размер перед парсингом.
Возвращает распарсенный объект или default.

dumps_compact(obj)
Сериализует в JSON без пробелов (компактный формат).
Возвращает строку.

is_json_like(text)
Проверяет начинается ли строка с { или [.
Возвращает bool.
